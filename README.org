* Overview

This repository is a declarative workstation configuration built with
[[https://github.com/yunfachi/denix][Denix]], [[https://github.com/nix-darwin/nix-darwin][nix-darwin]], and
[[https://github.com/nix-community/home-manager][Home Manager]]. It composes host definitions, reusable features, and selectable
desktop themes (rices) into system and user configurations.

Denix discovers =.nix= files under =hosts/=, =modules/=, and =rices/=. The
canonical ownership rules and change checklist are in
=docs/denix-architecture.md=.

* Getting Started

Enter the development shell, then use =just= to list the maintained workflows:

#+begin_src sh
nix develop
just
just check
just home
just switch
#+end_src

New files used by the flake must be git-tracked before evaluation. Prefer the
Nix-native workflows in this repository over ad hoc package state.

** Deploying the =seiran-vm1= Tart VM

On a Nix-equipped VM without =darwin-rebuild=, bootstrap it with one switch:

#+begin_src sh
sudo nix run nix-darwin/master#darwin-rebuild -- switch --flake .#seiran-vm1
#+end_src

Use the installed command for subsequent switches:

#+begin_src sh
sudo darwin-rebuild switch --flake .#seiran-vm1
#+end_src

Both commands activate the nix-darwin system and its integrated Home Manager
configuration.

* Repository Orientation

- =hosts/= contains machine facts and host-only bindings.
- =modules/= contains reusable program, service, shared-data, and toplevel integration modules.
- =rices/= contains desktop-experience policy selected through repository-owned interfaces.
- =docs/= contains stable cross-cutting policy and runbooks.
- =packages/= contains repository-owned package derivations.
